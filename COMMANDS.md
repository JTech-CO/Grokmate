# 명령어 (Commands)

## 호출 방법

- **웨이크워드** — 메시지 맨 앞에 `그록아`(한국어) 또는 `Hey Grok`(영어)
- **멘션** — `@Grokmate ...`
- **DM** — 봇에게 다이렉트 메시지
- **자연어 링크 읽기** — 링크에 "읽어 / 요약"을 덧붙이면 페이지를 가져와 요약
- **자연어 서버 관리(권한 필요)** — "슬로우모드 10초로", "역할 부여" 등

응답 언어는 웨이크워드를 따릅니다(`그록아`→한국어, `Hey Grok`→영어, 멘션·DM→한국어).

## 누구나 쓰는 슬래시 명령

| 명령 | 설명 |
|---|---|
| `/ping` | 봇 응답·지연 확인 |
| `/level [user]` | 레벨·XP 확인(대상 생략 시 본인) |
| `/leaderboard` | 서버 레벨 순위(상위 10) |
| `/verify-age` | 만 19세 이상 자가 인증(성인 채널 이용용, 본인만 표시) |
| `/repo <검색어>` | GitHub 공개 레포 검색(별점순 상위) |
| `/gh-user <유저명>` | GitHub 사용자 프로필·레포 목록 |
| `/gh-file <owner/repo> [경로]` | 레포 파일 내용·디렉터리 개요 |
| `/fetch <url> [ask]` | 웹 페이지를 읽어 요약(출처 포함) |

## 관리자·스탭 전용

> 권한이 없으면 명령 목록에 표시되지 않습니다.

| 명령 | 설명 |
|---|---|
| `/slowmode <초>` | 채널 슬로우모드(0–21600초) |
| `/timeout <user> <분> [사유]` | 타임아웃 |
| `/kick <user> [사유]` | 킥 |
| `/ban <user> [사유]` | 밴 (확인 버튼) |
| `/purge <개수>` | 메시지 청소 (확인 버튼) |
| `/nsfw-policy <on/off>` | 서버 성인 모드 활성/비활성 (확인 버튼) |

밴·킥·타임아웃·청소는 채팅(자연어)으로 실행하지 않고 **슬래시 명령 + 확인 버튼**으로만 실행됩니다. 슬로우모드·역할 부여/회수·채널 생성 등은 권한자가 자연어로도 지시할 수 있습니다.

모든 관리 액션은 **요청자 권한으로 단일 인가 게이트를 거치고 전수 감사 로그**에 기록됩니다.

1. 수집·처리하는 데이터

봇은 기능 제공에 필요한 최소한의 데이터만 처리합니다. 실명·생년월일·주소·전화번호·신분증 등 오프라인 신원 정보는 수집하지 않습니다.

2. 이용 목적

  • 이용자의 요청에 응답하고 봇 기능(대화·검색·코딩·이미지 분석·번역 등)을 제공하기 위해.
  • 관리 액션을 요청자의 권한으로만 실행하고 그 결과를 감사 기록으로 남기기 위해.
  • 성인 콘텐츠 게이팅과 미성년자 보호, 안전 필터(하드차단·인젝션 방어)를 적용하기 위해.
  • 요금제별 사용량 한도를 적용하고 결제를 처리하기 위해.
  • 서비스 오남용·악용을 탐지·차단하고 안정적으로 운영하기 위해.

수집한 데이터를 위 목적 외로 이용하지 않으며, 판매·광고 타게팅·제3자 마케팅 공유를 하지 않습니다.

3. 보관 기간과 저장 위치

  • 대화 맥락 및 일부 레이트리밋·쿨다운 상태 — 인메모리, 단기. 봇 재시작 시 소멸.
  • 서버 설정·연령 인증·레벨·경고·감사 로그·사용량·구독 — 봇 운영용 데이터베이스에 저장.
  • 위 데이터는 해당 기능이 유효한 동안 또는 목적 달성에 필요한 기간 동안 보관되며, 봇이 서버에서 제거되거나 이용자가 삭제를 요청하면 처리·보관을 중단·삭제합니다(감사·법적 의무상 최소 기간 보관이 필요한 항목 제외).

4. 제3자 처리·제공

기능 제공을 위해 일부 데이터가 아래 외부 서비스로 전송되어 처리될 수 있습니다. 각 서비스에는 해당 사업자의 개인정보 정책이 적용됩니다.

  • Discord — 봇이 동작하는 플랫폼. 메시지·상호작용은 Discord를 통해 오갑니다.
  • xAI (Grok API) — 응답·분석 생성을 위해 호출한 메시지 본문, 분석 대상 이미지, 검색 질의 등이 전송됩니다.
  • 웹 / X 검색 — 검색 모드 이용 시 검색 질의가 xAI 서버사이드 검색 도구로 전송됩니다.
  • GitHub API — /repo·/gh-user·/gh-file 이용 시 조회 요청이 전송됩니다.
  • 이용자가 지정한 URL — /fetch 또는 자연어 링크 읽기 이용 시 해당 페이지를 가져옵니다.
  • Stripe — 유료 플랜 결제 처리. 결제수단 정보는 Stripe가 직접 수집·보관하며 봇은 고객·구독 식별자만 받습니다.

이미지 번역(/translate)은 이미지를 외부에 보내지 않습니다. 이미지 속 글자는 봇 내부의 OCR(WASM, tesseract.js)로 추출한 뒤, 추출된 텍스트만 번역을 위해 xAI로 전송합니다. 이는 이미지 기반 프롬프트 인젝션을 줄이기 위한 설계입니다.

5. 국외 이전

위 제3자(특히 xAI·Stripe·GitHub·Discord)는 대한민국 외 국가(예: 미국)에 서버를 둘 수 있습니다. 봇을 이용하면 기능 제공에 필요한 범위에서 데이터가 국외로 이전·처리될 수 있음에 동의하는 것으로 간주됩니다. 각 사업자는 자체 보안·이전 보호조치를 적용합니다.

6. 보안

  • 시크릿(API 키·봇 토큰 등)은 코드·로그·저장소에 노출되지 않도록 관리합니다.
  • 모든 관리 액션은 요청자 권한 단일 인가 게이트를 거치고 전수 감사됩니다.
  • 외부 URL 가져오기는 사설/예약 IP·DNS 리바인딩 차단, robots.txt 확인, 사용자별 레이트리밋, 크기·시간·리다이렉트 상한 등의 가드를 거칩니다.
  • 결제 서명은 검증되며, 요금제(티어)는 조작 가능한 값이 아니라 실제 결제 정보로만 판정됩니다.

다만 인터넷을 통한 전송·저장에 절대적 보안은 없으며, 상용 수준의 합리적 보호조치를 적용함을 안내합니다.

7. 미성년자 보호

  • 성인(NSFW) 기능은 연령제한 채널 + 서버 활성 + 만 19세 이상 연령 인증 + 성적 의도의 4중 게이트를 모두 통과해야만 동작합니다.
  • 미성년자 관련 성적 콘텐츠는 어떤 게이트·모드·우회 시도에서도 예외 없이 차단됩니다.
  • 디스코드 이용약관상 만 13세 미만은 디스코드를 이용할 수 없습니다. 봇은 아동의 개인정보를 알면서 수집하지 않으며, 그러한 정보를 인지하면 삭제합니다.

8. 이용자의 권리

  • 본인 데이터의 열람·정정·삭제를 요청할 수 있습니다(아래 연락처).
  • 서버 관리자가 서버에서 봇을 제거하면 해당 서버 관련 데이터 처리는 중단됩니다.
  • 연령 인증 등 본인 관련 기록의 철회를 요청할 수 있습니다.
  • 요청 처리에는 본인·대상 확인이 필요할 수 있으며, 합리적 기간 내에 회신합니다.

9. 쿠키·소개 웹사이트

본 소개·문서 웹페이지(랜딩·약관·방침)는 정적 페이지로, 자체 추적 쿠키나 분석 스크립트를 심지 않습니다. 웹폰트 등 일부 자원은 외부 CDN에서 불러올 수 있으며, 이때 해당 제공자에게 표준적인 접속 정보(IP 등)가 전달될 수 있습니다.

10. 방침 변경

본 방침은 기능 추가·법령 변화 등에 따라 개정될 수 있으며, 변경 시 상단의 시행일·개정일을 갱신합니다. 중요한 변경은 합리적인 방법으로 안내하도록 노력합니다.

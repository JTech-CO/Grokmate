# 개인정보 처리방침 (Privacy Policy)

**시행일: 2026-07-08 · 최종 개정: 2026-07-08**

> 정식 게시본(HTML): [`legal/privacy.html`](legal/privacy.html). 이 문서는 GitHub 열람용 요약이며, 내용이 다를 경우 HTML 게시본이 우선합니다. 함께 [이용약관](legal/terms.html)도 확인하세요.

Grokmate("봇")가 디스코드에서 사용자 데이터를 어떻게 처리하는지 설명합니다. 봇을 서버에 추가하거나 봇과 상호작용하면 본 방침에 따른 처리에 동의하는 것으로 간주됩니다.

## 1. 수집·처리하는 데이터

실명·생년월일·주소·전화번호·신분증 등 오프라인 신원 정보는 수집하지 않습니다.

- **디스코드 식별자** — 사용자·서버(길드)·채널·메시지·역할 ID. 봇 동작·권한 판정·기능 연결에 사용.
- **메시지·콘텐츠** — 봇을 호출(웨이크워드/멘션/DM)한 메시지, 첨부 이미지·텍스트/코드 파일, `/fetch`·`/translate` 입력. 응답·분석 생성과 단기 대화 맥락 유지에 사용.
- **연령 인증 상태** — `/verify-age` 사용 시 "만 19세 이상" 자가 인증 여부(불리언)·시각·방식. **생년월일·신분증은 수집하지 않습니다.**
- **서버 설정** — 성인 모드 활성, 자동 모더레이션, 관리·스탭 역할, 환영·작별·자동역할·임시음성 허브 등.
- **모더레이션·경고** — 자동 모더레이션 로그, 경고 기록(대상·발부자·사유·시각), 관리 감사 로그(행위자·액션·대상·결과·시각).
- **활동·레벨** — 서버·사용자별 누적 XP·레벨.
- **사용량·토큰** — 일일 토큰 사용량(입력/출력), 메시지 수, 서버 단위 사용량·비용 메터. 요금제 한도는 **일일 토큰** 기준.
- **결제(선택)** — 요금제(Free/Pro/Max)·상태·유효 기간 종료 시각. **결제는 봇 밖(계좌이체·간편결제)에서 이루어지며, 봇은 카드·계좌 등 결제수단 정보를 수집·저장하지 않습니다.**
- **기능별 데이터** — 반응 역할 매핑, 임시 음성채널 소유권, 투표 및 투표 표(선택 항목).

**대화 맥락은 단기·인메모리로만 유지**됩니다(채널×사용자 단위, 최근 약 20개 메시지). 영구 저장하지 않으며 **봇 재시작 시 소멸**합니다.

## 2. 이용 목적

- 이용자 요청에 응답하고 봇 기능을 제공.
- 관리 액션을 요청자 권한으로만 실행하고 결과를 감사 기록.
- 성인 콘텐츠 게이팅·미성년 보호·안전 필터 적용.
- 요금제 한도 적용 및 결제 처리, 오남용 탐지.

위 목적 외로 데이터를 **판매·광고 타게팅·제3자 마케팅 공유하지 않습니다.**

## 3. 저장과 보관

- 대화 맥락·일부 레이트리밋 상태는 인메모리·단기(재시작 시 소멸).
- 서버 설정·연령 인증·레벨·경고·감사 로그·사용량·구독은 봇 운영용 데이터베이스에 저장.
- 봇이 서버에서 제거되거나 삭제 요청 시 처리·보관을 중단·삭제(감사·법적 의무상 최소 보관 항목 제외).

## 4. 제3자 처리

- **Discord** — 봇이 동작하는 플랫폼.
- **xAI (Grok API)** — 응답·분석 생성을 위해 메시지 본문·분석 이미지·검색 질의 등이 전송(xAI 정책 적용).
- **웹 / X 검색** — 검색 모드 사용 시 검색 질의 전송.
- **GitHub API** — `/repo`·`/gh-user`·`/gh-file` 사용 시.
- **사용자가 지정한 URL** — `/fetch`·자연어 링크 읽기 사용 시 해당 페이지를 가져옴.

유료 플랜 결제는 계좌이체·간편결제 등 봇 밖에서 이루어지며, 결제 데이터를 처리하는 제3자 결제대행사(PG)를 봇에 연동하고 있지 않습니다.

> **이미지 번역(`/translate`)은 이미지를 외부로 보내지 않습니다.** 봇 내부 OCR(WASM)로 글자만 추출한 뒤, 추출된 텍스트만 번역을 위해 xAI로 전송합니다.

## 5. 국외 이전

위 제3자(특히 xAI·GitHub·Discord)는 대한민국 외(예: 미국)에 서버를 둘 수 있으며, 기능 제공에 필요한 범위에서 데이터가 국외로 이전·처리될 수 있습니다.

## 6. 보안

- 시크릿(API 키·봇 토큰)은 코드·로그에 노출되지 않습니다.
- 관리 액션은 요청자 권한 단일 게이트 + 전수 감사.
- 외부 URL 가져오기는 사설/예약 IP·DNS 리바인딩 차단, robots.txt 확인, 레이트리밋, 크기·시간 상한 등 가드를 거칩니다.
- 결제 서명 검증, 요금제는 실제 결제 정보로만 판정.

## 7. 미성년자

- 성인 기능은 만 19세 이상 + 연령제한 채널 등 4중 게이트를 모두 통과해야만 동작.
- 미성년자 관련 성적 콘텐츠는 어떤 경우에도 차단.
- 디스코드 이용약관상 만 13세 미만은 디스코드를 사용할 수 없습니다.

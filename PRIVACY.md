# 개인정보 처리방침 (Privacy Policy)

**시행일: 2026-06-17**

Grokmate("봇")가 디스코드에서 사용자 데이터를 어떻게 처리하는지 설명합니다. 봇을 서버에 추가하거나 봇과 상호작용하면 본 방침에 따른 처리에 동의하는 것으로 간주됩니다.

## 1. 처리하는 데이터

- **디스코드 식별자** — 사용자 ID·서버(길드) ID·채널 ID. 봇 동작과 권한 판정에 사용.
- **메시지 내용** — 사용자가 봇을 호출(웨이크워드/멘션/DM)한 메시지. 응답 생성과 단기 대화 맥락 유지에 사용.
- **연령 인증 상태** — `/verify-age` 사용 시 "만 19세 이상" 자가 인증 여부(불리언)와 인증 시각·방식. **생년월일·신분증 등 신원 정보는 수집하지 않습니다.**
- **서버 설정** — 관리자가 설정한 값(성인 모드 활성 여부, 자동 모더레이션, 관리 역할 등).
- **활동 데이터** — XP·레벨(레벨링 기능).
- **관리 감사 로그** — 관리 명령(밴/킥/슬로우모드 등)의 행위자·액션·대상·결과·시각(책임 추적·오남용 방지용).

## 2. 사용 목적

- 사용자 요청에 응답하고 봇 기능을 제공.
- 관리 액션을 요청자 권한으로만 실행하고 결과를 기록.
- 성인 콘텐츠 게이팅 및 미성년 보호.

## 3. 저장과 보관

- **대화 맥락**은 단기·인메모리로만 유지됩니다(채널×사용자 단위, 최근 약 20개 메시지). 영구 저장하지 않으며 **봇 재시작 시 소멸**합니다.
- 연령 인증 상태·서버 설정·레벨·감사 로그는 봇 운영용 데이터베이스에 저장됩니다.
- 위 데이터는 봇이 서버에서 제거되거나 사용자가 삭제를 요청할 때까지 보관될 수 있습니다.

## 4. 제3자 처리

응답 생성·기능 제공을 위해 일부 데이터가 외부 서비스로 전송됩니다.

- **xAI (Grok API)** — 응답·분석 생성을 위해 메시지 내용 등이 전송됩니다(xAI 정책 적용).
- **웹 / X 검색** — 검색 모드 사용 시 검색 질의가 전송됩니다.
- **GitHub API** — `/repo`·`/gh-user`·`/gh-file` 사용 시.
- **사용자가 지정한 URL** — `/fetch`·자연어 링크 읽기 사용 시 해당 페이지를 가져옵니다.

위 목적 외에 데이터를 **판매·광고·제3자 공유하지 않습니다.**

## 5. 보안

- 시크릿(API 키·봇 토큰)은 코드·로그에 노출되지 않습니다.
- 외부 URL 가져오기는 사설/예약 IP·DNS 리바인딩 차단, robots.txt 확인, 레이트리밋, 크기·시간 상한 등 가드를 거칩니다.

## 6. 미성년자

- 성인 기능은 만 19세 이상 + 연령제한 채널 등 4중 게이트를 모두 통과해야만 동작합니다.
- 미성년자 관련 성적 콘텐츠는 어떤 경우에도 차단됩니다.
- 디스코드 이용약관상 만 13세 미만은 디스코드를 사용할 수 없습니다.

## 7. 사용자 권리

- 본인 데이터의 열람·삭제를 요청할 수 있습니다(아래 연락처).
- 서버에서 봇을 제거하면 해당 서버 관련 데이터 처리는 중단됩니다.

본 방침은 변경될 수 있으며, 변경 시 상단 시행일을 갱신합니다.
